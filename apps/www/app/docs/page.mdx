## Quick Start

<Steps>
### Create a new project

```bash
npx create-craft-app my-app
cd my-app
```

### Set up environment variables

```bash
cp .env.example .env
```

### Set up the database

```bash
pnpm db:push
```

### Run the development server

```bash
pnpm dev
```

</Steps>

Open [http://localhost:3000](http://localhost:3000) to see your app.
