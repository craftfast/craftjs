## Quick Start

<Steps>
### Clone the repository

```bash
git clone https://github.com/craftfast/craftjs.git
cd craftjs
```

### Install dependencies

```bash
pnpm install
```

### Set up environment variables

```bash
cp .env.example .env
```

### Set up the database

```bash
pnpm db:push
```

### Run the development server

```bash
pnpm dev
```
</Steps>

Open [http://localhost:3000](http://localhost:3000) to see your app.
